Thumbnail embed with external css

This shows how to point the player to an external css file and override the hard-coded thumb-embed button
The client / developer is responsible to serve this css file from a server that will not have CORS issues
Also, it is recommended to serve it with https to prevent insecure loading issues 

<div id="kaltura_player" style="width:400px;height:300px;"></div>
<script>
	kWidget.thumbEmbed({
		'targetId': 'kaltura_player',
		'wid': '_243342',
		'uiconf_id': '21099702',
		'entry_id': '0_uka1msg4',
		'flashvars':{
			"thumbnailUrl":"http://player.kaltura.com/docs/images/logo-145.png"
		}
	});
</script>
